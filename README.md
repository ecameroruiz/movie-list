Movie List
============

### Application

Run application
```
sh run_app.sh
```
This script will:
- Activate app environment
- Install all requirements
- Serve flask application on [http://localhost:8000/movies/](http://localhost:8000/movies/)

___

### Tests
Run tests
```
sh run_tests.sh
```
This script will:
- Activate app environment
- Install all requirements
- Run all unit tests
